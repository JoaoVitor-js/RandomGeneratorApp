# :construction_worker: How to run
Para rodar o projeto √© necessario um celular com o aplicativo do [expo](https://play.google.com/store/apps/details?id=host.exp.exponent) instalado ou um emulador android/ios.

```sh
git clone https://github.com/RafaelGoulartB/RandomNumberApp.git

cd RandomNumberApp/

npm install

npm start
```
Depois leia o codigo QRCode com o app do [expo](https://play.google.com/store/apps/details?id=host.exp.exponent) ou rode no emulator.
